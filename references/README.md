# References

Source PDFs, bibliography entries, and future transcriptions for the non-Shannon inequality program.

## Layout

```text
references/
  papers/          Source PDFs and other primary documents.
  transcriptions/  Verified or in-progress transcriptions.
  papers.bib       Canonical BibTeX entries for documentation and Lean docstrings.
```

## Current Focus

The initial bibliography is scoped to Track A and its immediate mathematical neighborhood:

- Zhang and Yeung on the first non-Shannon inequalities
- Dougherty, Freiling, and Zeger on the six DFZ inequalities
- Matús on infinite families and non-polyhedrality
- Chan and Yeung as background for the adjacent Track B correspondence
- Kaced and Romashchenko on essential conditional inequalities

The entries in `papers.bib` are the canonical citation keys for repository prose and paper-derived Lean docstrings.
